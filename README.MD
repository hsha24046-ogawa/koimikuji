# 恋みくじ(最強版)

ボタンをクリックするだけで今日の恋愛運を占うことができる、シンプルで楽しいWebアプリケーションです。

## 概要
このアプリは、HTML、CSS、JavaScriptの基礎を組み合わせて作成されました。
「おみくじを引く」ボタンを押すと、ランダムに運勢が表示されます。

## 特徴
- **シンプル操作**: ボタンを1回クリックするだけで結果が表示されます。
- **ランダム機能**: JavaScriptの数学関数 (`Math.random`) を使用して、毎回異なる結果を表示します。
- **レスポンシブデザイン**: ピンクを基調とした、親しみやすいデザインを採用しています。

## 構成ファイル
プロジェクトは以下の3つのファイルで構成されています。

- `index.html`: アプリの構造（タイトル、ボタン、結果表示エリア）を定義。
- `style.css`: ピンクの背景やボタンのアニメーションなど、見た目を装飾。
- `script.js`: おみくじのロジック（ランダムな運勢の抽選と表示）を制御。

## 使い方
1. このリポジトリ（またはファイル群）をダウンロードします。
2. `index.html` を任意のブラウザ（Google Chrome, Microsoft Edge, Safariなど）で開きます。
3. 「おみくじを引く」ボタンをクリックして、今日の運勢を確認してください。

## 運勢の種類
おみくじの結果には以下の4種類が含まれています：
- **大吉**: 素敵な出会いがあるかも！
- **中吉**: 少し勇気を出すといいことが起きる
- **小吉**: 今日はゆっくり過ごそう
- **凶**: 無理せず慎重に行動しよう

## 技術的なポイント
- **JavaScript**: `Math.floor(Math.random() * fortunes.length)` を使用して、配列の中からランダムに要素を選択しています。
- **CSS**: ボタンに `hover` エフェクトを追加し、マウスを重ねたときに色が濃くなるような演出を加えています。
